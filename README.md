# jTTPlayer 资源库

---

本项目纯用于`管理相关外部资源`。关于资源使用方式，请前往`jTTPlayer`项目。

## 版权声明

项目里的皮肤为移植版本，皮肤版权归原作者所有  



## 项目说明

- Skin目录：jTTPlayer皮肤资源及其相关，从原版.skn文件移植而来  

- Bass目录：jTTPlayer播放器内核及其相关，bass音频库、NativeBass（Java API）  

- Radio目录：电台相关文件，清单、.jttpl播放列表文件  



## 补充说明
- 电台播放：目前仅mpv内核支持
- .jttpl播放列表文件：播放列表 --> `列表（下拉）菜单` --> 添加 / 打开列表
