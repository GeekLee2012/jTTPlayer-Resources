# jTTPlayer 资源库

---

本项目纯用于`管理相关外部资源`。 

关于资源使用方式，请前往`jTTPlayer`项目，查阅相关文档。



## 版权声明

项目里的皮肤为移植版本，皮肤版权归原作者所有  



## 项目说明

- Skin目录：jTTPlayer皮肤资源及其相关，从原版.skn文件移植而来  

- Bass目录：jTTPlayer播放器bass内核及其相关，bass音频库、NativeBass（Java API）  

- mpv目录：jTTPlayer播放器mpv内核及其相关

- Radio目录：电台相关文件，清单、.jttpl播放列表文件、.m3u文件、.m3u8文件等



## 补充说明
- Windows平台mpv [mpv下载地址](https://sourceforge.net/projects/mpv-player-windows/files/release/)

- macOS平台，请前往`jTTPlayer`项目，查阅`Wiki.md`文档

- 电台播放：目前仅mpv内核支持

- .jttpl播放列表文件：播放列表 --> `列表（下拉）菜单` --> 添加 / 打开列表
